## ⚠️ This repository contains the backend of the Continuum project.

## Concept
Continuum is an app that provides customizable dashboards to help users visualize their progress and performance over time.  
It uses charts and activity heatmaps to make personal evolution measurable and easy to understand.

## Problem
Many people want to improve themselves, but lack simple and clear tools to consistently track their habits, goals, and long-term progress.

## Features
- Custom dashboards based on user-defined tags and goals  
- Activity heatmaps to visualize consistency over time  
- Charts showing progress and performance across different metrics
